This is a webapp designed to help students navigate campus life.
Our features to facilitate this will be:
-an event calendar that will allow users to bookmark items they're interested in
-an interactive map to show were on campus an event is taking place
-user profiles that save settings on how to filter the events they see
-comments section for an event

As a student, I want to see event locations on a map to know where i should go.

As a student, I want to bookmark events to check them out easily later.

As a student, I want to only see the events that are near me or related to my interests.

As a student, I want to be able to comment on events and see others comments/thoughts.
